# ChromaLens — 해커톤 갤러리 제출 자료

---

## 프로젝트 소개

ChromaLens는 URL 하나만 입력하면 해당 웹사이트의 색상 팔레트를 자동으로 추출·분석하는 도구입니다.
디자이너와 개발자가 레퍼런스 사이트의 색상 구성을 파악하거나, 자신의 프로젝트에 테마를 적용할 때 드는 반복 작업을 없애 줍니다.
추출된 색상은 카테고리별 팔레트, HSL 색상환, Before/After 테마 미리보기로 시각화되어, 단순한 색상 추출을 넘어 **디자인 의사결정 도구**로 활용할 수 있습니다.
Next.js 14 기반으로 구현되어 있으며, 배포 즉시 사용 가능한 완성된 웹 앱입니다.

---

## 기술적 강점

### 1. Puppeteer 실시간 추출
브라우저 자동화(Puppeteer)로 실제 렌더링된 DOM에서 CSS 속성을 직접 읽어냅니다.
정적 HTML 파싱이나 스크린샷 색상 샘플링 방식과 달리, JavaScript로 동적으로 적용된 스타일까지 정확히 캡처합니다.

### 2. Canvas 픽셀 기반 테마 교체
HSL 색상환을 HTML Canvas로 직접 렌더링하며, 픽셀 단위 연산으로 색조(Hue)·채도(Saturation)·명도(Lightness)를 실시간으로 조작합니다.
슬라이더 조작 즉시 Before/After 미리보기가 갱신되어, 색상 변경의 시각적 임팩트를 바로 확인할 수 있습니다.

### 3. 2-pass 카테고리 병합 알고리즘
추출된 수백 개의 색상을 단순히 나열하지 않고, 첫 번째 패스에서 색조 거리 기반으로 군집화하고 두 번째 패스에서 인접 카테고리를 병합합니다.
이를 통해 "비슷한 파란색 계열"이 하나의 카테고리로 묶이며, 실용적인 팔레트 분석 결과를 제공합니다.

이번 해커톤은 "실제로 쓸 수 있는 도구"를 만들어보자는 목표로 시작했습니다.

Puppeteer와 Canvas API를 조합하는 과정에서 예상보다 많은 엣지 케이스(투명 색상 처리, SSL 우회, 카테고리 병합 튜닝)를 만났지만, 그 덕분에 더 견고한 결과물이 나왔다고 생각합니다.
짧은 시간 안에 아이디어를 작동하는 앱으로 만들어내는 경험이 무척 값졌습니다.
